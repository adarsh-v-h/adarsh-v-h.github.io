High-Performance Limit Order Book

Modern C++ Low-Latency Custom Allocators Cache Tuning Concurrency

A functional financial matching engine optimized specifically for sub-microsecond tick-to-trade hot paths.

  • Architected using entirely zero-copy data structures and custom contiguous memory layouts to eliminate latency spikes.
  • Enforced strict cache-line alignment and explicit padding rules to completely eliminate false sharing and memory latency overhead.
  • Designed allocation-free execution paths to guarantee highly deterministic runtime performance under intense execution load.
  • Integrated a custom Data-Split Pool Allocator utilizing a Structure of Arrays (SoA) layout and Hot/Cold static data splitting (packing 16-byte OrderHot data separately from 24-byte out-of-band OrderCold metadata).
